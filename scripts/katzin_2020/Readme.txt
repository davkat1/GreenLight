Files for replicating the results of:
    Katzin, Van Mourik, Kempkes, Van Henten (2020). “GreenLight – An Open Source Model for Greenhouses with Supplemental
    Lighting: Evaluation of Heat Requirements under LED and HPS Lamps.” Biosystems Engineering 194: 61–81.
    https://doi.org/10.1016/j.biosystemseng.2020.03.010.

Which also appeared as:
    Katzin (2021). “Energy Saving by LED Lighting in Greenhouses: A Process-Based Modelling Approach.”
    PhD thesis, Wageningen University, 2021. https://doi.org/10.18174/544434. Chapter 3.

In this folder:
    katzin_2020_format_input_data.py        Format input data required for running the simulations
    katzin_2020_run_sims.py                 Run the simulations as in Katzin 2020
    katzin_2020_generate_output.py          Generate the output as in Katzin 2020
    _katzin_2020_generate_figures.py        Generate the figures of Katzin 2020. Used by katzin_2020_generate_output.py
    _katzin_2020_generate_tables.py         Generate the tables of Katzin 2020. Used by katzin_2020_generate_output.py
    _katzin_2020_analysis_functions.py      Functions for analyzing the data of Katzin 2020. Used by katzin_2020_generate_tables.py
