Files for replicating the results of:
    Katzin, Marcelis, Van Mourik (2021). “Energy Savings in Greenhouses by Transition from High-Pressure Sodium to LED Lighting.”
    Applied Energy 281: 116019. https://doi.org/10.1016/j.apenergy.2020.116019.

Which also appeared as:
    Katzin (2021). “Energy Saving by LED Lighting in Greenhouses: A Process-Based Modelling Approach.”
    PhD thesis, Wageningen University, 2021. https://doi.org/10.18174/544434. Chapter 4.

In this folder:
    katzin_2021_format_input_data.py        Format input data required for running the simulations
    katzin_2021_run_sims.py                 Run the simulations as in Katzin 2021
    katzin_2021_generate_output.py          Generate the output (Figure 6) as in Katzin 2021, and compare current results to it
